## Database of Zeolite Structures

**Contains 243 structures, as of 2025-02-11**

This file contains zeolite structures, in [CIF format](https://en.wikipedia.org/wiki/Crystallographic_Information_File), taken in 2025 from the IZA database at https://www.iza-structure.org/databases/ (the Database of Zeolite Structures maintained by the Structure Commission of the International Zeolite Association).

There is no clear license attached to the structures, but they are published and widely shared in the academic community.
